# TrackMint 🍃

A simple and easy-to-use personal expense tracker that works on both your web browser and Android phone. I built this to help track everyday spending, set goals, and see where money goes without any complicated setups. 

## What it does

- **Home Dashboard:** Quickly see your recent expenses and income at a glance.
- **Easy Logging:** Adding a new expense or income is fast and simple.
- **Goal Tracking:** Set savings goals and watch your progress over time.
- **Charts & Stats:** Simple charts show exactly what you're spending your money on.
- **History:** See a full list of all your past transactions.
- **100% Private:** Your data stays on your device. You don't even need an internet connection to use it.
- **Mobile Ready:** Works perfectly on phones and can be built into a real Android app.

## What I used to build it

- **React** for the user interface
- **Vite** for fast building
- **Capacitor** to turn it into an Android app
- **Vanilla CSS** for styling
- **Local Storage** to save data privately

## How to run it on your computer

1. Download the code:
   ```bash
   git clone https://github.com/malshan444/TrackMint.git
   cd TrackMint
   ```

2. Install the packages:
   ```bash
   npm install
   ```

3. Start the app:
   ```bash
   npm run dev
   ```

## How to build it for Android

If you want to run it on your Android phone using Android Studio:
1. Build the project: `npm run build`
2. Sync the files: `npx cap sync android`
3. Open Android Studio: `npx cap open android`
